Cevap Anahtarı:

- Açıklayıcı yorumlar
- Alternatif çözüm örnekleri

KAPSANAN KONULAR

1. Import ve Temel İşlemler
   - Kütüphane kullanımı (math modülü)
   - Temel matematiksel işlemler

2. Print Fonksiyonu
   - Formatlı çıktı üretme
   - sep ve end parametreleri
   - f-string kullanımı

3. Değişkenler ve Veri Tipleri
   - Değişken tanımlama
   - Veri tipi kontrolü
   - Tip dönüşümleri

4. Listeler
   - Liste oluşturma ve erişim
   - Liste metodları (append, remove, sort, vb.)
   - Liste dilimleme ve indeksleme

5. Karakter Dizileri (Strings)
   - String işlemleri
   - String metodları (split, upper, lower)
   - String birleştirme ve formatlama

6. Sözlükler (Dictionaries)
   - Sözlük oluşturma ve erişim
   - Sözlük metodları (get, items, pop, copy)
   - Anahtar-değer çiftleriyle çalışma

7. Operatörler
   - Aritmetik operatörler (+, -, *, /, //, %, **)
   - Mantıksal operatörler (and, or, not)
   - Karşılaştırma operatörleri

8. Koşullu İfadeler
   - if/elif/else yapıları
   - Koşullu program akışı
   - Pratik uygulamalar (not hesaplama, sayı kontrolü)

9. Döngüler
   - for döngüsü
   - while döngüsü
   - range() fonksiyonu
   - İç içe döngüler

10. Fonksiyonlar
	- Fonksiyon tanımlama
	- Parametreler ve return değerleri
	- Liste ve sözlük işleyen fonksiyonlar

11. Karmaşık Alıştırmalar
    - Liste filtreleme
    - Kelime sayma
    - En büyük sayıyı bulma
    - Sözlük istatistikleri
